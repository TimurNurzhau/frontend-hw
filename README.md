YouTalk Blog

Адаптивная вёрстка блога психологического сервиса YouTalk.

📱 О проекте

Проект выполнен в рамках учебного задания по frontend-разработке. Реализованы две страницы:
Блог — главная страница со списком статей и фильтрацией
Статья — страница отдельной публикации с боковой панелью

🛠 Технологии

- HTML5
- CSS3
- Flexbox / CSS Grid
- Адаптивная вёрстка (5+ брейкпоинтов)
- БЭМ-нейминг
- JavaScript (бургер-меню)

📂 Структура проекта
frontend_hw/
├── index.html # стартовая страница
├── blog.html # главная блога
├── article.html # страница статьи
├── css/ # стили (6 файлов)
├── js/ # скрипты
├── icons/ # SVG-иконки
├── images/ # PNG-изображения
└── fonts/ # шрифты Montserrat + Open Sans



🎯 Реализовано

- ✅ Семантическая вёрстка
- Использованы теги: `<header>`, `<main>`, `<footer>`, `<article>`, `<aside>`, `<nav>`, `<section>`, `<figure>`, `<figcaption>`, `<blockquote>`, `<time>`, `<cite>`
- Правильная иерархия заголовков: `<h1>` → `<h2>` → `<h3>` → `<h4>`
- На каждой странице только один `<h1>`
- Хлебные крошки реализованы через `<nav>` с `aria-label`

- ✅ БЭМ-именование классов
- Формат: `block__element--modifier`
- Примеры: `.post-card__title`, `.post-card--expanded`, `.main-nav__link--has-dropdown`
- Отсутствует вложенность селекторов глубже 3 уровней
- Модификаторы используются для вариаций блоков

- ✅ Адаптив под все устройства (320px → 1440px)
- 1024px — Сайдбар сужается (282px → 222px), скрываются декоративные блоки.
- 768px — Появляется бургер-меню, основное меню и кнопка "Подобрать психолога" скрываются.
- 480px — Футер перестраивается в колонку (универсальные правки для всех мобильных устройств).
- 414px — Сетка карточек перестраивается с 6 колонок на 1 колонку.
- 390px — Правки футера и типографики для iPhone 12/13/14 Pro.
- 375px — Правки футера и типографики для iPhone SE, iPhone 12/13/14 mini.
- 320px — Минимальные отступы, уменьшенные шрифты (8-14px), специальные правки для футера

- ✅ Сетка карточек (CSS Grid)
- Десктоп (от 1024px и выше): Сетка состоит из 6 колонок. Большие карточки занимают 3 колонки (grid-column: span 3), маленькие карточки — 2 колонки (grid-column: span 2). Отступы между карточками составляют 24px.
- Планшет (768px – 1023px): Сетка перестраивается на 2 колонки. Отступы между карточками уменьшаются до 20px.
- Мобильные устройства (до 414px): Сетка перестраивается на 1 колонку. Отступы между карточками составляют 16px.

- ✅ Бургер-меню на мобильных устройствах
- Бургер-меню появляется на экранах шириной 768px и меньше. При клике на кнопку-бургер открывается основное меню и кнопка "Подобрать психолога". Меню фиксированное, имеет тень для визуального отделения от контента. Закрытие меню происходит при повторном клике на бургер-кнопку. Позиция открытого меню адаптируется под высоту шапки на разных разрешениях.

- ✅ Интерактивность и эффекты
- Hover-эффекты: при наведении на пункты меню цвет текста меняется на зелёно-бирюзовый (#03b2a5); кнопки затемняются; заголовки карточек статей подчёркиваются; фильтры получают цветную обводку; иконки социальных сетей меняют прозрачность.
- Псевдоэлементы: ::before используется для создания точек-маркеров в списках и иконки кавычек в цитатах. ::after используется для стрелки в пункте меню "Материалы" и разделителей в метаданных статьи.
- Псевдоклассы: активно используются :hover, :focus-visible, :last-child, :not(:last-child).

- ✅ Доступность 
- alt-атрибуты присутствуют у всех изображений. В атрибутах прописаны описательные тексты для скринридеров.
- aria-label используется для навигации (<nav>), кнопки бургера и кнопки подписки в подвале.
- aria-hidden="true" добавлен для всех декоративных элементов (розовые блики, подарочные иконки, декоративные изображения).
- aria-current="page" присутствует в хлебных крошках на каждой странице, указывая текущую позицию.
- visually-hidden — специальный класс для визуального скрытия элементов, которые остаются доступными для скринридеров.
- lang="ru" прописан в корневом элементе <html>.
- focus-visible добавлен для всех интерактивных элементов, обеспечивая видимый фокус при клавиатурной навигации.
